# Event Planner App — SIT305 Task 4.1P - Deakin University 

Android app built lets you add, view, edit and delete personal events that are saved locally on the device.

## Features
- Add events with title, category, location and date/time
- View all events sorted by date
- Edit or delete existing events
- Confirmation dialog before deleting
- Inline validation errors for empty fields and past dates
- Data persists after app close and device restart

## Tech used
- Java, Android Studio
- Room database for local storage
- Jetpack Navigation for fragment switching
- Material Design 3 components
- RecyclerView for the event list

## How to run
1. Clone the repo
2. Open in Android Studio
3. Let Gradle sync
4. Run on emulator or device (API 26+)
